Usage Guide

KEY Mode

1. Press "Start" to activate listeners

2. Press F3 to save mouse coordinates

3. Press F1 to execute configured clicks

4. Clicks execute ONCE per F1 press

MOUSE Mode

1. Press "Start" to activate listeners

2. Click with mouse to execute clicks

3. Each mouse click executes the full sequence

AUTO Mode

1. Press "Start" to begin continuous execution

2. DelayClicks execute in automatic loop

3. Press F2 to stop

F1 Execute clicks (KEY mode only)
F2 Stop AutoClick (all modes)
F3 Save coordinates
